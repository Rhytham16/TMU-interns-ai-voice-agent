# Project Overview: TMU Interns AI Voice Agent

This project is an AI-powered voice agent developed as part of the TMU Interns program. It uses speech recognition, natural language understanding, and vector-based search to assist users through voice interactions.

### Features
- Voice-based input and output
- FastAPI backend for managing API requests
- Document loading and vector embedding via LangChain and ChromaDB
- Modular architecture

### Tech Stack
- Python
- FastAPI
- LangChain
- ChromaDB
- OpenAI / LLMs
